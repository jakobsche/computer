public property TChipTempSensor.TripPoints[I: Integer] : TTripPoint
  read GetTripPoints;

Description

Ein gültiger Index ergibt einen gültigen Schaltpunkt (Reaktion des Kernels auf Erreichen der Schaltpunkttemperatur). Seine Bedeutung ergibt sich aus den weiteren Eigenschaften der TTripPoint-Komponente mit dem angegebenen Index

Errors

Ein ungültiger Index der TTripPoint-Komponente löst Exceptions aus, wenn andere Eigenschaften des Schaltpunkts abgefragt werden
